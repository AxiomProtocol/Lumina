LUMINA SECURITY POLICY

REPORTING VULNERABILITIES

Do not report security vulnerabilities in public issues.

To report a vulnerability, please send an email to: info@axiomprotocol.app

INFORMATION TO INCLUDE

When reporting a security vulnerability, please include the following information:

(a) A clear description of the vulnerability.

(b) Steps to reproduce the issue.

(c) The potential impact of the vulnerability.

(d) Any suggested remediation or fix, if available.

RESPONSE

We will acknowledge receipt of your report and will work to investigate and address the issue. We appreciate responsible disclosure and will keep you informed of our progress.

QUESTIONS

For questions about this security policy, contact: info@axiomprotocol.app
